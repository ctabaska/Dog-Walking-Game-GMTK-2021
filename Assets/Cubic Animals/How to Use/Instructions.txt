All the animals have 4 frames which must be combined as follows:


frame 0 + 2 = Idle

frame 0 + 1 = Walk

frame 1 = Jump

frame 2 = Crouch

frame 3 = Hit / Defeated


The gif in this file is used as an example of the animations
